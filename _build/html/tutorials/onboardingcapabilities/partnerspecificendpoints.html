Points de terminaison spécifiques aux partenaires¶

Le portail d’intégration de Contentserv permet à l’utilisateur (détaillants/fournisseurs) d’avoir des points de terminaison spécifiques aux partenaires pour importer leurs données dans le système. Avoir des points de terminaison uniques garantit que les erreurs de données sont identifiées et mappées pour les partenaires respectifs, et permet ainsi d’améliorer la qualité globale des données.

Conditions préalables¶

  • Partenaires
  • Utilisateurs avec des rôles définis

Termes clés¶

  • Points de terminaison - Le point de terminaison est le processus final où nous mappons les partenaires avec le processus défini. Ce point de terminaison sera visible sur le tableau de bord du partenaire.
  • Flux de travail - Le flux de travail définit la structure et les composants d’un fichier qui doit être importé dans le système ou exporté hors du système. Il fournit tous les détails sur tous les composants configurés dans une feuille. Le système interprétera le fichier importé selon la manière dont nous en définissons les détails dans cette section.
  • Mappage - La section mappage permet de mapper les différents éléments (attributs, balises, classes et taxonomies) définis dans un fichier d’intégration par rapport à leurs éléments respectifs (attributs, balises, classes et taxonomies) définis dans le système.

Étapes pour configurer les points de terminaison¶

  1. Connectez-vous à l’application.

  1. Cliquez sur l’icône “Administration”.

  1. Accédez au menu Intégration de données -> Points de terminaison.


  1. Cliquez sur l’icône “+” pour créer le point de terminaison.

  1. Renseignez les informations demandées comme Nom, Type et Code et cliquez sur le bouton CRÉER.


  1. Renseignez les informations demandées comme MAPPAGE EN TEMPS RÉEL, SYSTÈME, FLUX DE TRAVAIL, MAPPAGE, ONGLET TABLEAU DE BORD, CATALOGUE PHYSIQUE et cliquez sur le bouton ENREGISTRER.


  1. Accédez au menu Administration des partenaires -> Partenaires.

  1. Cliquez sur le partenaire créé auquel vous souhaitez affecter le point de terminaison.

  1. Sélectionnez le point de terminaison créé dans la section Systèmes en cliquant sur le point de terminaison.

  1. Cliquez sur l’onglet Rôles.

  1. Sélectionnez le point de terminaison créé dans la section Systèmes en cliquant sur le point de terminaison.

  1. Maintenant, cliquez sur l’icône “Explorer” et allez dans Intégration de données.

  1. Cliquez sur le lien Téléchargement de fichiers du point de terminaison créé pour télécharger le fichier souhaité.
