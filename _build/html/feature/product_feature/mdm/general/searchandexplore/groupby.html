Group By¶

The Group By feature allows a user to view a group of objects in the system by specifying an attribute.

Description:
  • The feature provides a user the functionality to view a grouping of products with the desired sequence of attributes and tags (filterable).
Related topics:
  • Group By to make groups according to Tags and Attributes.
Application:

  • The feature provides a user the functionality to view a grouping of products with the desired sequence of attributes and tags (filterable).

  • The feature allows a user to specify the sequence of grouping attributes.

  • Example:

    If a user wants to view a group of products that is related to Female section, the Group By feature may be used.

How to do: Group By¶

To do a group by, follow these steps:


  1. Click the Group By button found on the upper right corner of the screen.


  1. Click the “+” icon to add the properties of a grouping. A list of all the filterable properties will be shown.


  1. Select the property/properties to be used. A list of values will be displayed along with the number of products.



  1. View the products that are displayed after selecting a value.


  1. To add another level for the grouping, click the “+” icon.


  1. Similarly, on the list that will appear, select the appropriate property. Select a subtag to have a more refined grouping.
