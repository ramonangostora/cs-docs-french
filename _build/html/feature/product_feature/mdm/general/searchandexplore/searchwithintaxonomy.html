Search within Taxonomy¶

The Search within Taxonomy feature allows a user to search (for an article or a product) or to navigate within the system.

Description:
  • This feature allows a user to search for an item or to navigate the system.
  • This feature provides a classification of products that gives a user the ease to search.
  • This feature provides a user a hierarchy of objects in the Taxonomy that can be used for navigation.
Related topics:
  • Parametric Search
  • Multi-faceted Search
  • Group By
Application:

  • This feature is used to search for a product using the taxonomy.

  • Example:

    If a user wants to search for an object or navigate the system, the user can use the Taxonomy Hierarchy.

How to do: Search within Taxonomy¶

To do a search within taxonomy, follow these steps:


  1. Click the Choose Taxonomies button found on the left side of the Search bar.


  1. From the pop-up menu, select the appropriate taxonomy (e.g. Products). Children nodes are also displayed.


  1. Tick on the checkbox, then click the Apply button to display the products that belong to a specific node.



  1. Click on a node to display its children nodes (along with the respective product count). The selected node is highlighted in bold text.


  1. Tick on the checkbox of the title heading to select all children nodes at once.


  1. To view the entire taxonomy tree structure including the blank nodes, click the Show All button.



  1. Click the Clear Selection button to reset all selected taxonomy nodes.
