Manage LOVs¶

Manage LOVs lets you create a property with many possible values.

Description:
  • Classify your products by setting up LOVs as tags.
  • Unlike categories, there is no hierarchy in tags.
Related topics:
  • Manage Affinities
Application:
  • Have a property where only one value applies to the product.
  • Multi-select the values that are applicable to the product.
  • Apply a boolean tag when the value is either true or false.

How to do: Manage LOVs¶

Easily classify your products by using LOVs as tags, and here’s how to do it.


  1. Click the Gear icon to go to Settings.


  1. Check if the Tag Configuration tab is displayed.


  1. Click on the “+” icon to create a new tag (LOVs).


  1. Check if a new tag is created at the top of the list.


  1. Enter the name of the tag and select the type of tag from the drop-down list.


  1. Enter the other values such as Description, Color, and Multiselect then click Save.


  1. Click the Create Tag Value button found at the far right column.


  1. Enter the label and click on the search tab then click Save.



  1. Repeat the above steps for other tag values.
