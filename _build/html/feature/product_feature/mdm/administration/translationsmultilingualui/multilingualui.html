Interface utilisateur multilingue¶

Interface utilisateur multilingue est une fonctionnalité de traduction des éléments de menu de l’interface utilisateur, afin que celle-ci soit utilisable dans plusieurs langues.

Terme clé:
  • Interface utilisateur (UI): L’interface numérique utilisée pour interagir avec le logiciel Contentserv.
Description:
  • L’interface utilisateur peut être affichée en différentes langues comme le français, l’allemand et l’espagnol.
  • L’administrateur peut configurer le modèle de données (attributs, balises, classes) en différentes langues afin qu’il soit compris par les utilisateurs de plusieurs pays ou régions.
  • La configuration de cette fonctionnalité s’effectue depuis la section d’administration.
Thèmes associéss:
  • Modèle de données multilingue
Application:

  • Le système peut afficher l’interface utilisateur en plusieurs langues.

  • Exemple:

    L’administrateur souhaite que l’interface utilisateur soit en français pour un utilisateur français. L’administrateur configure les traductions des libellés de l’interface utilisateur dans la section d’administration. Lorsque l’utilisateur sélectionne sa langue, l’interface utilisateur du système est traduite.

Marche à suivre: Gestion d’une interface utilisateur multilingue¶

Pour ce faire, procédez comme suit:


  1. Allez dans Administration.


  1. Cliquez sur l’onglet Traductions.


  1. Cliquez sur Libellés de l’application.


  1. Selon la langue requise, saisissez les traductions du libellé concerné (dans cette démonstration, nous avons sélectionné « Accélération »).


  1. Déconnectez-vous et reconnectez-vous.




  1. Sur la page de connexion, sélectionnez la langue désirée à partir de la barre d’outils en haut à droite de votre écran.
