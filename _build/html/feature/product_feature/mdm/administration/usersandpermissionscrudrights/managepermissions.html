Manage Permissions¶

Manage Permissions lets you define what the user can or cannot do.

Description:
  • Simply assign the user to the role with the appropriate functions and say goodbye to unnecessary system changes.
  • Have full control as to who gets to view, edit, and delete the products in the system.
Related topics:
  • Manage View Templates
Application:

  • User can access only the permitted functions.

  • Example:

    Let’s say you did not allow a certain user to delete products, then that user will not have the delete function.

How to do: Manage Permissions¶

Choose who gets to do what or not, and here’s how to do it.


  1. On the home page, click the Gear icon to go to Settings.


  1. Click the Permissions Configuration button.


  1. On the left pane, select a role.



  1. Select a class and set the permission for Create, Read, Update, and Delete. Do the same for taxonomy. A green circle indicates that the function is enabled for that item.
