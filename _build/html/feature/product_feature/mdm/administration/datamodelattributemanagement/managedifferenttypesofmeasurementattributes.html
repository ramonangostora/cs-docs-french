Measurement Type: Volume¶

This feature example allows an administrator to set an attribute as Volume.

Description:
  • This feature allows the setting of an attribute type as Volume.
  • This feature makes an attribute type as Volume and applies the appropriate default units to be associated with the volume.
Related topics:
  • Volume type attribute.
Application:

  • Volume measurement type can set the volume of a liquid product (e.g. Beverages) and also set its corresponding unit.

  • Example:

    An administrator can set an attribute of a Red Bull product to volume so that a user can easily enter the volume of the product and select a unit (from a drop-down list).

How to do: Measurement Type: Volume¶

To set an attribute as a volume, follow these steps:


  1. Go to Settings.


  1. Go to the Attribute Configuration section.


  1. Click the “+” icon to create an attribute.


  1. Select Volume under the type field.


  1. Select the appropriate unit for the volume attribute (e.g. Pint, Quart, Cup, and more).


  1. Set the Precision and the Default Value.


  1. Click on the Save button to save the settings.
