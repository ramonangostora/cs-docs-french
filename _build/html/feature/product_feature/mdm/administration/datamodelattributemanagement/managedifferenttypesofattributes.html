Manage Different Types of Attributes¶

The Manage Different Types of Attributes is a feature that allows an administrator to manage and set up attributes in the system.

Description:
  • This feature can create and manage different types of attributes (properties) for various data. This will restrict the attribute to the selected type only.
Related topics:
  • Types of Attribute to be selected in the configuration.
Application:

  • The feature will allow an administrator to set up the attributes in the system that will help other users put relevant information to the product data fields.

  • Example:

    An administrator sets up a Date type attribute for a data field. If a general user adds an information to that field, a calendar will pop up and the user can select a date value using the calendar.

How to do: Manage Different Types of Attributes¶

To do this admin feature, follow the steps below:


  1. Go to Settings.


  1. Go to the Attribute Configuration section.


  1. Click the “+” icon to create an attribute.


  1. For a particular attribute type, refer to its respective section below:

Type: Text

  1. Enter the attribute name and select the type as Text.


  1. View the text attribute in runtime.



Type: Number

  1. In the configuration, enter the attribute name and select the type as Number.


  1. View the number attribute in runtime.



Type: Price

  1. In the configuration, enter the attribute name and select Currency in the type field.


  1. Select the desired currency value for the default unit.


  1. Set the precision.


  1. Provide the default value.


  1. View the price attribute in runtime.



Type: Date

  1. In the configuration, enter the attribute name and select Date in the type field.


  1. Set also the values for Default Value, Placeholder, and Description, if available.
