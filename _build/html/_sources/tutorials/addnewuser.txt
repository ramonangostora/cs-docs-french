================================
Ajouter de nouveaux utilisateurs
================================
Gérer différents rôles ou profils dans le système en ajoutant de nouveaux utilisateurs.


Comment ajouter de nouveaux utilisateurs
=========================================
Pour le faire, suivez les étapes suivantes:

|

#.  Cliquez sur **Settings** (paramètres).

.. image:: /feature/_images/mdmimages/addingnewusers_clicksettings.PNG
|
|


#.  Cliquez sur **Partner Admin** (admin des partenaires).

.. image:: /feature/_images/mdmimages/addingnewusers_clickpartadmin.PNG
|
|


#.  Cliquez sur **Users** sur la barre de gauche.

.. image:: /feature/_images/mdmimages/addingnewusers_clickusers.PNG
|
|


#.   Ensuite, cliquez sur « + » pour ajouter un utilisateur.

.. image:: /feature/_images/mdmimages/addingnewusers_clickplus.PNG
|
|


#.   Entrez les données dans les différents champs.

.. image:: /feature/_images/mdmimages/addingnewusers_enterdetails.PNG
|
|


#.  Ensuite, cliquez sur **Create** (créer).

.. image:: /feature/_images/mdmimages/addingnewusers_enterdetailsandcreate.PNG
|
|


#.   Afficher l’utilisateur ajouté.

.. image:: /feature/_images/mdmimages/addingnewusers_addeduser.PNG
|
|


Vous pouvez aussi ajouter une icône pour l’utilisateur. Pour cela, suivez les étapes suivantes:

      #.    Cliquez sur l’icône **Add** et sélectionnez une image sur votre ordinateur.

      .. image:: /feature/_images/mdmimages/addingnewusers_clickaddicon.PNG
      |
      |

      #.    Afficher l’icône ajoutée.

      .. image:: /feature/_images/mdmimages/addingnewusers_iconadded.PNG
      |
      |

|
