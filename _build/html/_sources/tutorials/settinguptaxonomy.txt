========================
Configurer une Taxonomie
========================
La taxonomie est une classification de produits utilisée pour organiser le contenu.


Comment configurer une taxonomie
================================
Pour le faire, suivez les étapes suivantes:

#. Cliquez sur **Settings** (paramètres).

.. image:: /feature/_images/mdmimages/settinguptaxonomy_clicksettings.PNG
|
|

#. Cliquez sur l’onglet **Data Model**.

.. image:: /feature/_images/mdmimages/settinguptaxonomy_clickdatamodel.PNG
|
|

#.  Cliquez sur **Taxonomies** pour afficher le contenu.

.. image:: /feature/_images/mdmimages/settinguptaxonomy_expandtaxonomy.PNG
|
|

#.   Ensuite, cliquez sur **Master Taxonomy**.

.. image:: /feature/_images/mdmimages/settinguptaxonomy_clickmastertaxonomy.PNG
|
|

#.   Utilisez l’icône « + » pour afficher le contenu, verticalement ou horizontalement.

.. image:: /feature/_images/mdmimages/settinguptaxonomy_useplus.PNG
|
|

#.   Une fenêtre Taxonomie apparaîtra. Entrez les données de cette nouvelle taxonomie et cliquez sur Create (créer).

.. image:: /feature/_images/mdmimages/settinguptaxonomy_taxdetails.PNG
|
|

Des icônes peuvent également être ajoutées à la taxonomie.

    #. Cliquez sur la section sous taxonomie (dans cette démo, nous sélectionnons **Tops**).

    .. image:: /feature/_images/mdmimages/settinguptaxonomy_clickcollarless.PNG
    |
    |

    #.   Cliquez sur l’icône de chargement et sélectionnez une image sur votre ordinateur.

    .. image:: /feature/_images/mdmimages/settinguptaxonomy_clickaddicon.PNG
    |
    |

   #.   Afficher l’icône sélectionnée.

    .. image:: /feature/_images/mdmimages/settinguptaxonomy_iconadded.PNG
    |
    |
