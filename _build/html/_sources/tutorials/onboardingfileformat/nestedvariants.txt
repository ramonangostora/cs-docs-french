====================================================
Variantes imbriquées dans un fichier d'intégration
====================================================


* L'ensemble des variantes imbriquées de tous les produits seront répertoriées dans une seule feuille d'un fichier Excel.

*	La feuille des variantes imbriquées contiendra les détails suivants pour chaque variante imbriquée par colonnes:


=========================   =======================================================================================================================================================================================================================================================================
Détails                      Description
=========================   =======================================================================================================================================================================================================================================================================
ID de variante imbriquée     Vous devez mentionner l'identifiant unique de chaque variante imbriquée dans cette colonne. Il n’y a aucune restriction sur l’en-tête de colonne/le nom de colonne.

ID d'article                 Vous devez mentionner l'identifiant unique d'un article pour lequel vous définissez la variante imbriquée dans cette colonne. Il aura la valeur de la colonne **ID d'article** de la feuille dans laquelle vous avez défini tous les articles.

ID de variante               Vous devez mentionner l'identifiant unique de la variante pour laquelle vous définissez la variante imbriquée dans cette colonne.

Variante                     Vous devez mentionner le nom de la variante comme en-tête de colonne pour laquelle vous définissez la variante imbriquée. Par exemple, si vous souhaitez définir une variante d'unité imbriquée, définissez l'en-tête de colonne comme **Unité**. Si vous définissez une variante d'unité, vous devez définir des sous-types de variante d'unité dans cette colonne pour laquelle vous souhaitez définir une variante imbriquée.

Variante imbriquée           Vous devez mentionner le nom de la variante imbriquée comme en-tête de colonne. Par exemple, si vous souhaitez définir GTIN comme variante imbriquée, alors définissez l'en-tête de colonne comme **GTIN**. Vous devez indiquer la valeur GTIN dans cette colonne.
=========================   =======================================================================================================================================================================================================================================================================
