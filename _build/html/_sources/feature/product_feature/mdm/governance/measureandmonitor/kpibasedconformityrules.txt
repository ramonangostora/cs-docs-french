===================================
Règles de conformité selon les KPI
===================================
Règles de conformité selon les KPI est une fonctionnalité de gouvernance des données qui définit des règles pour évaluer la conformité d'un *Article* ou d'un *Produit* aux normes admises.


====================     =======================================================================================================================================================================
**Terme clé:**           * **Indicateur de performance clé (KPI):** est une mesure utilisée pour déterminer la qualité des données relative aux produits qui sont ajoutées dans le système en lien avec une norme établie.

**Description:**         * Les KPI contribuent à déterminer l'exactitude des produits selon les attributs, taxonomie et balises d'un article.
                         * Les KPI valident les 4 facteurs suivants: *Intégralité, Conformité, Unicité et Exactitude*.
                         * Les KPI permettent un approfondissement sur la base des *taxonomie* et *balises*.
                         * Le KPI: **Conformité** vérifie l'exactitude des données du produit et inspecte également que les données respectent les normes.

**Thèmes associés:**     * :doc:`Gestion des balises </feature/product_feature/mdm/administration/datamodeltagmanagement/index>`
                         * :doc:`Gestion des attributs </feature/product_feature/mdm/administration/datamodelattributemanagement/index>`
                         * :doc:`Gestion des taxonomies </feature/product_feature/mdm/administration/simpletaxonomytree/managetaxonomy>`
                         * :doc:`Suivi de la qualité des données </feature/product_feature/mdm/governance/measureandmonitor/monitordataquality>`

**Application:**         * **La Conformité des données** est l'une des caractéristiques primordiales du contrôle de la qualité et de l'exactitude des données. C'est un facteur essentiel aux KPI.
                         * *Exemple:*

                              Une organisation vend des meubles. Ils présentent des attributs pour *Longueur du bois* et des balises pour *Couleur*.
                              Ensuite, la *Règle de conformité* aide les utilisateurs à vérifier que la longueur ne doit pas excéder 400 m (40 000 cm).

====================     =======================================================================================================================================================================


Marche à suivre: Règles de conformité selon les KPI
====================================================
Pour ce faire, procédez comme suit:

|

#. Connectez-vous et allez dans *Administration*.

.. image:: /feature/_images/mdmimages/cs_monitordataquality_login.PNG
|
|


#. Cliquez sur l'onglet **Gouvernance des données**.

.. image:: /feature/_images/mdmimages/cs_monitordataquality_clickondatagovernance.PNG
|
|


#. Cliquez sur **KPI**.

.. image:: /feature/_images/mdmimages/cs_monitordataquality_clickonkpis.PNG
|
|


#. Cliquez sur l'icône "+" pour créer un nouveau KPI.

.. image:: /feature/_images/mdmimages/cs_monitordataquality_clickontheicontocreatenewkpi.PNG
|
|


#. Renseignez le nom du KPI et cliquez sur le bouton **Créer**.

.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_filloutthename.PNG
|
|


#. Une fois créé, le nouveau KPI apparaîtra dans la liste. Cliquez sur le bouton **Modifier** à droite.

.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_oncethekpiiscreated.PNG
|
|


#. Sur la page *Détails du KPI*, sélectionnez une *Classe* et une *Taxonomie*, qui utiliseront le KPI. Cliquez sur le bouton **Appliquer**.

.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_selectclassandtaxonomy.PNG
|
.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_selectclassandtaxonomy2.PNG
|
|


#. Pour approfondir le niveau de détail, ajoutez des *Balises ou des Taxonomies*, dans lesquelles le KPI sera classé.

.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_forthedrilldownlevels.PNG
|
|


#. Cliquez sur l'onglet **Conformité** et sur le bouton **+ Ajouter une règle** pour créer une nouvelle règle.

.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_clickontheconformity.PNG
|
|


#. Renseignez les détails pour *les Libellé, Attributs et Balise* auxquels la règle sera appliquée. Sélectionnez les règles selon les exigences et les normes.

.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_providethedetails.PNG
|
.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_providethedetails2.PNG
|
|


#. Cliquez sur le bouton **Enregistrer**.

.. image:: /feature/_images/mdmimages/cs_kpibasedconformity_clickthesave.PNG
|
|


*Remarque: Une notification vous confirmera que la règle et le KPI ont été créés. Cliquez sur OK pour continuer.*

|
|
