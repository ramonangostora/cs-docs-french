================
Édition en bloc
================
La fonctionnalité Édition en bloc permet à l'utilisateur de modifier les valeurs de plusieurs attributs/balises pour plusieurs produits en une seule opération.


=====================     ===============================================================================================================================================================
**Description:**          * Édition en bloc permet de modifier les valeurs de plusieurs attributs/balises pour plusieurs produits en une seule opération.
                          * Édition en bloc peut être utilisée sur des types de produits similaires ou différents.
                          * Édition en bloc permet de modifier plusieurs attributs/balises en une seule opération.
                          * Édition en bloc permet de modifier plusieurs attributs/balises et de les appliquer aux produits.
                          * Édition en bloc est accessible sur la page principale du catalogue PIM.

**Thèmes associés:**      * :doc:`Rechercher dans Taxonomie </feature/product_feature/mdm/general/searchandexplore/searchwithintaxonomy>`
                          * :doc:`Regrouper par </feature/product_feature/mdm/general/searchandexplore/groupby>`

**Application:**          * Édition en bloc est utilisée pour réaliser différentes actions, notamment: *Édition, Étiquetage, Changement d'état, Déplacement ou Suppression*.
                          * Édition en bloc permet à l'utilisateur d'éviter plusieurs opérations manuelles en éditant les produits dans un lot plutôt que de procéder produit par produit.
                          * Édition en bloc permet de modifier et de définir un vaste éventail d'attributs de produit tels que *Prix, Comparer au Prix, Poids, Balises* et plus encore.
                          * *Exemple:*

                               Une entreprise agroalimentaire a modifié le slogan ou la description de ses produits.
                               L'entreprise peut sélectionner ses produits dans le système et modifier la description grâce à Édition en bloc.
                               La fonctionnalité appliquera aussi le changement de description à tous ses produits.

=====================     ===============================================================================================================================================================


Marche à suivre: Édition en bloc
=================================
Pour exécuter une opération d'édition en bloc, procédez comme suit:

|

#. Sélectionnez les éléments à modifier en cliquant sur la case à cocher miniature.

.. image:: /feature/_images/mdmimages/bulkedit_selectitems.PNG
|
|


#. Cliquez sur le bouton **Édition en bloc**.

.. image:: /feature/_images/mdmimages/bulkedit_clickonthebulkeditbutton.PNG
|
|


#. Un menu déroulant contenant une liste d'attributs et de balises s'ouvre.

.. image:: /feature/_images/mdmimages/bulkedit_viewthepopupmenu.PNG
|
|


#. Vous pouvez soit (1) sélectionner les balises/attributs dans la liste, soit (2) rechercher les balises/attributs dans la barre de recherche en saisissant les valeurs. Reportez-vous aux images ci-dessous.


  #. Sélection des balises/attributs dans la liste:

  .. image:: /feature/_images/mdmimages/bulkedit_selectingtags.PNG
  |
  |

  #. Recherche des balises/attributs dans la barre de recherche:

  .. image:: /feature/_images/mdmimages/bulkedit_searchingtags.PNG
  |
  |

|


#. Cliquez sur le bouton **Appliquer** pour exécuter l'Édition en bloc des produits.

.. image:: /feature/_images/mdmimages/bulkedit_clickontheapplybutton.PNG
|
.. image:: /feature/_images/mdmimages/bulkedit_clickontheapplybutton2.PNG
|
|


#. Consultez les modifications appliquées par l'édition en bloc.

.. image:: /feature/_images/mdmimages/bulkedit_viewappliedchange.PNG
|
.. image:: /feature/_images/mdmimages/bulkedit_viewappliedchange2.PNG
|
|
