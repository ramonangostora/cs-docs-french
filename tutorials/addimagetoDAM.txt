========================
Ajouter une image au DAM
========================
Les produits avec des images sont plus attrayants pour les utilisateurs du système, vous pouvez les stocker dans le DAM (gestionnaire des actifs). Pour le faire, suivez les étapes suivantes:

|


#. Cliquez sur **All**.

.. image:: /feature/_images/mdmimages/addingimage_clickall.PNG
|
|


#. Cliquez sur **DAM**.

.. image:: /feature/_images/mdmimages/addingimage_clickDAM.PNG
|
|


#.  Passez la souris sur l’icône « + ».

.. image:: /feature/_images/mdmimages/addingimage_hoverplus.PNG
|
|


#.   Ensuite, cliquez sur **Image**.

.. image:: /feature/_images/mdmimages/addingimage_selimage.PNG
|
|


#.   Cliquez sur l’icône **Add Image** (ajouter une image) dans la section « File Preview » (aperçu) du fichier et sélectionnez l’image souhaitée sur votre ordinateur.

.. image:: /feature/_images/mdmimages/addingimage_clickcloud.PNG
|
|


#.   L’image s’affichera alors dans la section « File Preview » du fichier et vous n’aurez plus qu’à donner un nom à cette image.

.. image:: /feature/_images/mdmimages/addingimage_entername.PNG
|
|


#.  Cliquez ensuite sur **Save** (enregistrer).

.. image:: /feature/_images/mdmimages/addingimage_enternamesave.PNG
|
|


#.  Proposez une meilleure expérience produit à vos clientsCliquez sur les **Metadata** (métadonnées).

.. image:: /feature/_images/mdmimages/addingimage_clickmeta.PNG
|
|


#.	Les métadonnées disponibles dans la section Métadonnées seront générées automatiquement, mais les champs peuvent aussi être remplis manuellement.

.. image:: /feature/_images/mdmimages/addingimage_compmeta.PNG
|
|
