========================
Créer un nouveau produit
========================
Le système permet une gestion efficace des produits. Les produits peuvent être créés, supprimés et modifiés.

Comment créer de nouveaux produits
==============================
Pour le faire, suivez les étapes suivantes:


|

#.	Cliquez sur  **All**  haut à droite.

.. image:: /feature/_images/mdmimages/createnewprod_clickall.PNG
|
|

#. Passez la souris sur l’icône « + » et sélectionnez le produit souhaité (dans cette démonstration, nous sélectionnons **un seul produit**).

.. image:: /feature/_images/mdmimages/createnewprod_hoverplus.PNG
|
|

.. image:: /feature/_images/mdmimages/createnewprod_clicksingleprod.PNG
|
|

#. Entrez les informations nécessaires. Les champs en rouge doivent être remplis.

.. image:: /feature/_images/mdmimages/createnewprod_completeattrib.PNG
|
|

#. Cliquez sur **Add Taxonomy** pour ajouter une taxonomie au produit.

.. image:: /feature/_images/mdmimages/createnewprod_addtax.PNG
|
|

#. Le menu Taxonomie apparaîtra. Sélectionnez la taxonomie souhaitée (dans cette démo, nous sélectionnons **Tops**).

.. image:: /feature/_images/mdmimages/createnewprod_taxmenu.PNG
|
|

.. image:: /feature/_images/mdmimages/createnewprod_seltax.PNG
|
|

#.  Consulter la taxonomie sélectionnée

.. image:: /feature/_images/mdmimages/createnewprod_taxselected.PNG
|
|

#.  Consulter la taxonomie sélectionnéeCliquez sur l’onglet **Relationships** pour ajouter une image au produit.


.. image:: /feature/_images/mdmimages/createnewprod_clickrel.PNG
|
|

#.  Cliquez sur « + » pour ajouter une entité.


.. image:: /feature/_images/mdmimages/createnewprod_caddentity.PNG
|
|

#.  Cliquez sur le bouton de sélection pour ajouter l’image souhaitée (le bouton devient bleu lorsque vous le sélectionnez), puis cliquez sur **Add**.

.. image:: /feature/_images/mdmimages/createnewprod_imageselect.PNG
|
|

.. image:: /feature/_images/mdmimages/createnewprod_addall.PNG
|
|

#.  Vous pouvez aussi faire glisser l’image sélectionnée dans le volet de droite pour ajouter l’image.

.. image:: /feature/_images/mdmimages/createnewprod_dragimage.PNG
|
|

#.  Pour ajouter d’autres images, cliquez sur **Return** et répétez les étapes précédentes.

.. image:: /feature/_images/mdmimages/createnewprod_clickret.PNG
|
|
