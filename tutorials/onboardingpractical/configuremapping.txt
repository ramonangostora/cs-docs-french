======================
Configurer le mappage
======================

|

#. Connectez-vous à l'application et cliquez sur l'icône "Administration".

.. image:: /tutorials/_images/cs_createuser_logintotheapp.PNG
|
|


#.	Accédez au menu **Intégration de données -> Mappages**.

.. image:: /tutorials/_images/cs_configuremapping_gotodataintegration.PNG
|
|


#.	Cliquez sur l'icône "+" pour créer les mappages.

.. image:: /tutorials/_images/cs_configuremapping_clicktheicontocreate.PNG
|
|


#.	Renseignez les informations demandées comme **Nom** et **Code** et cliquez sur le bouton **CRÉER**.

.. image:: /tutorials/_images/cs_configuremapping_providethedetails.PNG
|
|


#.	Cliquez sur l'icône "Editer" à côté du mappage.

.. image:: /tutorials/_images/cs_configuremapping_clickontheeditbutton.PNG
|
|


#.	Une fenêtre contextuelle s'affiche à l'écran. Cliquez sur l'onglet **Attribut**.

.. image:: /tutorials/_images/cs_configuremapping_onthepopupwindow.PNG
|
|


#.	Cliquez sur le bouton **Ajouter un mappage** pour mapper les attributs définis dans un fichier d'intégration avec les attributs système.

.. image:: /tutorials/_images/cs_configuremapping_clickontheaddmapping.PNG
|
|


#.  Définissez le nom d'attribut utilisé dans un fichier d'intégration dans la zone de texte à gauche et sélectionnez l'attribut système équivalent dans le champ **Élément mappé**.

.. image:: /tutorials/_images/cs_configuremapping_definetheattributename.PNG
|
|


#.	De même, vous pouvez mapper des balises, des taxonomies et des classes définies dans un fichier d'intégration avec les balises, taxonomies et classes du système dans les onglets intitulés Balise, Taxonomie et Classe, respectivement.

.. image:: /tutorials/_images/cs_configuremapping_likewisemaptags.PNG
|
|


#.	Cliquez sur le bouton **ENREGISTRER**.

.. image:: /tutorials/_images/cs_configuremapping_clickonthesavebutton.PNG
|
|


Configurer le mappage - Éléments de l’écran
============================================

=========    =================================     ============================================================
N° série      Éléments de l'écran                   Description
=========    =================================     ============================================================
1             Nom                                   Indiquez le nom du mappage.

2             Code                                  Indiquez le code.

3             CRÉER                                 Cliquez sur le bouton CRÉER pour créer le mappage.

4             Onglet Attribut                       Définissez l'attribut utilisé dans une feuille d'un fichier d'intégration dans la zone de texte à gauche et sélectionnez l'attribut système équivalent dans le champ **Élément mappé**.

5             Onglet Balise                         Définissez la balise utilisée dans une feuille d'un fichier d'intégration dans la zone de texte à gauche et sélectionnez la balise système équivalente dans le champ **Élément mappé**.

6             Onglet Taxonomie                      Définissez la taxonomie utilisée dans une feuille d'un fichier d'intégration dans la zone de texte à gauche et sélectionnez la taxonomie système équivalente dans le champ **Élément mappé**.

7             Onglet Classe                         Définissez la classe utilisée dans une feuille d'un fichier d'intégration dans la zone de texte à gauche et sélectionnez la classe système équivalente à partir du champ **Élément mappé**.

8             ENREGISTRER                           Cliquez sur le bouton **ENREGISTRER** pour sauvegarder les détails du mappage.

9             IGNORER                               Cliquez sur le bouton IGNORER pour ignorer les détails du mappage défini.

=========    =================================     ============================================================
