==================================================
Définir des actifs dans un fichier d'intégration
==================================================


* Tous les actifs seront répertoriés dans une seule feuille d'un fichier Excel.

* Tous les actifs seront répertoriés par lignes dans une feuille. Chaque actif aura une ligne distincte dans une feuille.

* La feuille contiendra les détails suivants de chaque actif par colonnes:

====================   =======================================================================================================================================================================================================================================================================
Détails                 Description
====================   =======================================================================================================================================================================================================================================================================
ID de l'actif           Mentionnez l'identifiant unique de chaque produit dans cette colonne. Il n’y a aucune restriction sur l’en-tête de colonne/le nom de colonne.

Nom de l'actif          Mentionnez le nom de chaque actif dans cette colonne. Il n’y a aucune restriction sur l’en-tête de colonne/le nom de colonne.
====================   =======================================================================================================================================================================================================================================================================
